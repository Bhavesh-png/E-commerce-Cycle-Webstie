# 🚲 Patil Adventures – E-Commerce Cycle Website

**Patil Adventures** is a modern **front-end e-commerce website** designed for showcasing and selling **premium electric bicycles (e-bikes)** online.
The platform allows users to explore different bicycle brands, view detailed product specifications, and manage their selections through a dynamic shopping cart interface.

This project demonstrates **frontend web development skills using HTML, CSS, and JavaScript**, while creating a clean and responsive product showcase similar to a real online store.

---

# 🌐 Project Overview

This website simulates a **complete online cycle store experience** where users can:

* Browse various **electric bicycle brands**
* Explore **product specifications and highlights**
* Add cycles to a **dynamic shopping cart**
* Navigate through a **modern responsive interface**
* Search and filter products easily

The project focuses on **UI design, JavaScript interaction, and e-commerce layout structure**.

---

# ✨ Key Features

### 🚲 Product Catalog

* Multiple electric bicycle brands including:

  * **Firefox Electric**
  * **Hero Lectro**
  * **E-Motorad**

### 🛒 Dynamic Shopping Cart

* JavaScript-powered shopping cart
* Uses **localStorage** to store selected items
* Automatically calculates **total price**
* Displays a **purchase summary**

### 📦 Product Specifications

* Detailed information including:

  * Battery capacity
  * Motor type
  * Riding range
  * Charging time
  * Key highlights

### 🔎 Search Functionality

* Built-in **search bar** to quickly find specific bicycle brands.

### 🎨 Modern UI

* Dark theme interface
* Smooth hover animations
* Glassmorphism navigation bar
* Interactive design elements

### 📱 Responsive Design

* Fully responsive layout for:

  * Desktop
  * Tablet
  * Mobile devices

### 🔔 Interactive Notifications

* Toast notifications when items are **added to the cart**

# 🛠️ Technologies Used

### Frontend

# 🚀 Installation & Run Locally

Clone the repository:

```bash
git clone https://github.com/Bhavesh-png/E-commerce-Cycle-Webstie.git
```

Go to the project directory:

```bash
cd E-commerce-Cycle-Webstie
```

Run the project:

Open **index.html** in your browser.

For the best development experience:

1. Open the folder in **VS Code**
2. Install **Live Server extension**
3. Click **Go Live**

This project is a **static frontend website**, so no backend server is required.
